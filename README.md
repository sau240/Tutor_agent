Shiksha Mitra is a lightweight AI agent designed to help students with personalized learning. It functions smoothly without internet access by using locally running Gemma models. It offers structured, interactive, and visual explanations across various topics.

Key Features  
1. Multimodal Learning Support  
- Generates diagrams, visuals, and organized notes  
- Produces tables, summaries, and step-by-step solutions  
- Offers quiz mode, practice mode, and scoring  

2. Intelligent Image Understanding  
- Reads text from notebook photos (OCR)  
- Detects shapes, objects, and diagrams  
- Solves handwritten math problems  
- Explains images in simple language  

3. Offline AI Processing (Gemma / Ollama)  
- Works completely offline with Gemma 2B/7B models  
- Designed for rural and low-bandwidth areas  
- Provides faster responses without needing the internet  

4. Personalized Learning Agent  
- Remembers past questions for better answers  
- Stores student queries and responses in MongoDB  
- Develops a learning profile over time  
- Supports English, Hindi, and various Indian languages

Tech Stack  
- Frontend: React  
- Backend: Flask  
- AI Models: Gemma (2B/7B via Ollama), OpenAI for images  
- Database: MongoDB  
- Deployment: Offline + Hybrid  
- Features: Multimodal reasoning, organized outputs, memory for context
